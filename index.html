Project 2

Full Stack Amazon Web App

Using ReactJS, tailwind CSS, & Redux to create the frontend and Firebase for backend. Server side rendering handeled by NextJS. NextAuth for authentication and Stripe for checkouts. Created a webhook to verify stripe event & adding its details to the Firebase database. Deployed on Vercel.

Project 3

Google Docs Web App

Client side UI created using ReactJS & Tailwind CSS. Using Firebase state monitoring to enable realtime multi user editing. Implemented a richtext editor to edit documents. All of the document information is preserved and saved to Firebase firestore. NextAuth was used for user authentication. Deployed on Vercel.
